# 이정표농업 MVP 기획 메모

## 1. 서비스 핵심

- 고객 프로필에 따라 농업 지원사업을 추천한다.
- 추천 결과와 관련 있는 기업 광고를 함께 노출한다.
- 장기적으로는 공고 수집, 정리, 검수, 광고 운영을 관리자 페이지에서 관리한다.

## 2. 핵심 사용자

- 예비 청년농
- 지역 농업인 및 농업 법인
- 스마트팜 도입 희망 농가
- 농업 장비/시설/컨설팅 기업 광고주
- 운영 관리자

## 3. MVP 화면

- 메인 랜딩 페이지
- 맞춤 추천 입력 폼
- 추천 지원사업 결과 리스트
- 추천 광고 기업 카드
- 에이전트 운영 구조 소개 섹션

## 4. 다음 구현 우선순위

- 실제 공고 데이터를 저장할 데이터 스키마 설계
- 관리자 로그인/관리자 대시보드 설계
- 광고주 신청 폼 및 승인 프로세스 추가
- 회원가입 및 맞춤 추천 저장 기능 추가
- 정부/지자체 데이터 수집 파이프라인 연결

## 5. 추천 기술 방향

- 초기: 정적 프론트엔드 + JSON 데이터
- 다음: Next.js 또는 React 프론트엔드 + API 서버 + DB
- 운영: 관리자 페이지, 수집 배치, 광고 분석 리포트

## 6. MVP 에이전트 체계

- 기획 총괄 에이전트: 서비스 범위, 핵심 사용자, 광고 모델, 운영정책 정의
- 지원사업 리서치 에이전트: 정부/지자체/농업센터 공고 출처 발굴 및 수집
- 데이터 정리 에이전트: 공고를 지역, 대상, 분야, 기간, 금액, 조건 기준으로 표준화
- 추천 로직 에이전트: 고객 프로필 기반 적합도 계산과 추천 근거 설계
- 풀스택 구현 에이전트: 웹사이트, 관리자 페이지, API, 데이터 저장 구조 구현
- 보안 운영 에이전트: 회원/관리자 권한, 입력 검증, 로그, 보안정책 관리

## 7. 확장 에이전트

- UX/UI 디자인 에이전트: 신뢰감 있는 정보 구조와 모바일 사용성 개선
- 광고 운영 에이전트: 광고주 등록, 승인, 노출 위치, 성과 관리
- 품질검수 에이전트: 만료 공고, 잘못된 링크, 잘못된 추천 결과 점검
- 운영 자동화 에이전트: 정기 수집, 상태 갱신, 관리자 대기열 자동화
- 데이터 분석 에이전트: 클릭, 신청 전환, 광고 성과, 이탈률 분석

## 8. MVP 보안 최소 기준

- HTTPS 전면 적용
- 비밀번호는 Argon2 또는 bcrypt 해시 저장
- 관리자와 일반회원 권한 완전 분리
- 검색창과 모든 폼 입력값 서버 검증
- SQL Injection 방지를 위한 ORM 또는 prepared statement 사용
- 출력 데이터 HTML escape 처리로 XSS 방어
- 로그인/변경 폼에 CSRF 방어 적용
- 세션 쿠키에 HttpOnly, Secure, SameSite 설정
- 광고 신청 파일 업로드 확장자, MIME, 크기 제한
- 개인정보는 지역, 관심분야, 법인 여부 등 최소 항목만 수집
